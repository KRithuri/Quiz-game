Quiz Game

Quiz game is an interactive game that sets to test the user/player on programming concepts. Enjoy!

Quiz Guide

1. Player gets 10 set of questions. 2. Player must answer one question at a time. 3. Player gets to see score at the top. 4. Correct answer will be highlighted in green, incorrect with red after selection. 5. Final score will be displayed at the end as a percentage.
